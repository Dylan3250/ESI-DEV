Lotto

Implémentation d'un jeu de lotto sous la forme de pronostics et d'un tirage.

Les pronostics sont constitués de size chiffres différents. Les chiffres du pronostic sont tous compris entre 1 et max, ces valeurs comprises. Les pronostics sont implémentés par la classe nvs::lotto::Pronostic.

Le tirage est constitué selon les mêmes contraintes que les pronostics. Il est implémenté dans la classe nvs::lotto::Draw.

Les chiffres d'un pronostics sont choisis et fourni au constructeur. Les chiffres du tirage sont déterminés aléatoirement lors de sa construction.

Les pronostics pour un tirage donné sont rassemblés, avant qu'il ait lieu, dans la classe nvs::lotto::Lotto. Celle-ci s'occupe également de provoquer un tirage. Il est ensuite possible de connaître les pronostics corrects pour un nombre donnés de chiffres compris entre 1 et size.

La fonction nvs::lotto::data produit les données pour construire les pronostics.
