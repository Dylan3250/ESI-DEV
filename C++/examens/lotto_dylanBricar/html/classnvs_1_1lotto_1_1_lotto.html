Description détaillée

Classe gérant un jeu de lotto.

À sa création, on spécifie la valeur maximale à pronostiquer / tirer au sort, max, les chiffres vont de 1 à max et le nombre de chiffres d'un pronostic / tirage.

Ensuite on enregistre les pronostics.

Puis le tirage au sort des la combinaison gagnante est réalisé.

Enfin, on détermine les gagants.

On y spécifie le domaine des chiffres et le nombre de chiffres d'un pronostic / du tirage d'une grille de lotto.

Paramètres
sizele nombre de chiffres d'un pronostic / du tirage.
maxla valeur maximale que peuvent prendre les chiffres d'un pronostic / du tirage.
Exceptions
std::invalid_argumentdans les trois cas suivants :
  • le nombre de chiffres du tirage est nul ;
  • la valeur maximale possible du tirage est nulle ;
  • la valeur maximale possible du tirage est strictement inférieure au nombre de valeurs de celui-ci.

std::vector<Pronostic> nvs::lotto::Lotto::winner ( unsigned  n,
Equality  type = Equality::STRICT 
) const

Accesseur en lecture des pronostics gagnants.

Cette méthode ne peut être invoquée qu'une fois le tirage réalisé. Elle retourne l'ensemble des pronostics comportant :

  • exactement n chiffres présents dans le tirage lorsque la valeur de type est Equality::STRICT ;
  • au moins n chiffres présents dans le tirage lorsque la valeur de type est Equality::INCLUSIVE.
Paramètres
nle nombre du chiffres du pronostic présents dans le tirage.
typele type d'égalité utilisée (voir ci-dessus).
Renvoie
l'ensemble des pronostics répondant aux paramètres.
Exceptions
std::logic_errorsi le tirage n'a pas encore été réalisé.
std::invalid_argumentsi n est nul ou strictement supérieur au nombre de chiffres du tirage.
